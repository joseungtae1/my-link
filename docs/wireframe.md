# 마이링크 (MyLink) - 기능적 와이어프레임 (Wireframe)

복잡한 탭 메뉴나 모달창 없이, 텍스트(표시 이름 및 내용) 자체를 클릭해 조작하는 '인라인 에디팅' 중심의 대시보드 레이아웃입니다. 프로필 이미지와 테마 설정은 존재하지 않습니다.

## 1. 관리자 대시보드 (Admin Dashboard)

### 구조 (Desktop View 기준)
```text
+-------------------------------------------------------------+
| [로고 MyLink]                                    [내 계정 ▼] | <- 심플한 GNB 상단 고정
+-------------------------------------------------------------+
|                                  |                          |
|  [ 편집 영역 (디스플레이 60%) ]  | [ 미리보기 뷰어 (40%) ]  |
|                                  |                          |
|  ▶ 프로필 (인라인 에디팅)        |  +--------------------+  |
|  (마우스를 올려 클릭 시 input)   |  | [ 모바일 목업 프레임]|  |
|                                  |  |                    |  |
|  [ ✏️ 김크리 ]                  |  |     김크리         |  |
|  [ ✏️ 안녕하세요. 김크리입니다 ] |  |  안녕하세요. 김크리..|  |
|                                  |  |                    |  |
|  ------------------------------  |  |  +--------------+  |  |
|  ▶ 링크 리스트 (인라인 에디팅)   |  |  |(파비콘) 링크1 |  |  |
|                                  |  |  +--------------+  |  |
|  +----------------------------+  |  |  +--------------+  |  |
|  | [파비콘] [✏️ 타이틀 적기]   |  |  |  |(파비콘) 링크2 |  |  |
|  | URL: [✏️ https://...] [삭제]|  |  |  +--------------+  |  |
|  +----------------------------+  |  |                    |  |
|                                  |  |                    |  |
|  +----------------------------+  |  |      Powered by    |  |
|  | [+] 새로운 링크 추가 (빈칸)   |  |  |       MyLink       |  |
|  +----------------------------+  |  +--------------------+  |
+----------------------------------+--------------------------+
```

## 2. 방문자용 랜딩 페이지 (User Public Page)
`domain.com/닉네임` 형태로 제공되는 심플한 방문자 전용 랜딩 페이지입니다. (단일 디자인 적용)

```text
+---------------------------+
|                           |
|                           |
|       김크리 (닉네임)     | <- H1 (명확하고 모던한 폰트 적용)
|   안녕하세요 김크리입니다 | <- P
|                           |
|  +---------------------+  |
|  | [파비콘]   링크 1   |  | <- 구글 API파비콘 이미지와 Title
|  +---------------------+  | 
|                           |
|  +---------------------+  |
|  | [파비콘]   링크 2   |  |
|  +---------------------+  |
|                           |
|                           |
|      Powered by MyLink    |
+---------------------------+
```
