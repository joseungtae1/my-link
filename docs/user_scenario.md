# 마이링크 (MyLink) - 사용자 시나리오 (User Scenario)

## 1. 핵심 타겟 페르소나
- **김크리 (28세)**: 복잡한 관리 툴이나 꾸미기를 귀찮아하며, 가입 후 즉시 텍스트만 치면 예쁘게 최적화된 심플한 링크 모음 페이지를 얻고자 함.

## 2. 주요 유저 플로우

### 2.1 사이트 진입 및 로그인
1. 마이링크 접속 후 **[Google로 시작하기]** 로그인 완료.

### 2.2 대시보드 진입 및 닉네임 설정
1. 가입 후 가장 먼저 자신의 주소로 사용될 닉네임을 설정. (예: `kimcre`). 이 순간부터 `mylink.com/kimcre` 접속 주소를 소유하게 됨. 핸들(@)은 사용하지 않음.

### 2.3 프로필 인라인 편집 (Inline Edit)
1. 대시보드 상단에 위치한 자신의 닉네임이나 빈 "소개글을 입력해주세요" 텍스트를 마우스로 가볍게 클릭함.
2. 팝업 창이 뜨는 대신 해당 텍스트 영역 자체가 즉시 텍스트박스로 변함(인라인 편집 모드). "반갑습니다. 크리에이터 김크리입니다" 라고 치고 바깥 영역을 누르거나 엔터를 쳐서 즉석으로 반영시킴. (별도의 프로필 이미지 업로드는 없음)

### 2.4 링크 추가 및 데이터 수정 (Google API 파비콘)
1. 링크 리스트 영역에서 **[+ 새 링크 추가]** 가이드 텍스트를 클릭.
2. 즉석에서 타이틀("내 유튜브")과 URL(`https://youtube.com/...`)을 키보드로 치고 완료함.
3. 저장 완료와 거의 동시에 링크 목록 왼쪽에 Google S2 API를 통해 로딩된 유튜브 파비콘 아이콘이 깔끔하게 표시됨.
4. 나중에 오타를 발견했다면 다시 그 글자를 클릭해서 인라인으로 수정하고 완료함.

### 2.5 프로필 공유 및 랜딩 페이지
1. 우측 상단의 링크 복사 버튼을 눌러 본인의 `mylink.com/kimcre` 주소를 인스타그램 프로필 링크에 복붙함.
2. 외부 방문자가 접속했을 때, 별도의 테마 없이 깔끔한 화이트/블랙톤의 미니멀한 단일 디자인 템플릿에 이름, 소개, 파비콘이 예쁘게 놓인 링크 버튼들을 볼 수 있음.
